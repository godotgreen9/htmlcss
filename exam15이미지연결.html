자주 사용하는 속성들

alt 속성

화면에 이미지를 표시하거나, 이미지가 표시되지 않을 때 해당 이미지에 대한 간단한 설명을 달아주는 태그 웹 브라우저에 화면 낭독시(스크린 리더)를 통해 화면의 내용을 읽어 올 때 사용된다

width와 height 속성

사진의 높이와 길이를 지정하는 속성들이다 사진의 크기는 고정된 크기와 가변 크기로 설정 할 수 있다

고정 크기로 설정할 경우

화면(웹브라우저)의 크기가 변해도 이미지의 크기는 변하지 않는다

가변 크기로 설정할 경우

화면(웹브라우저)의 크기가 변해도 이미지의 크기도 변경된다

map

이미지에 구역을 설정하여 각 구역을 링크나 특정 작업을 연결하는 태그
